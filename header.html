O que é a tag HEADER?

Publicado em: 22 de Maio de 2025

A tag <header> é um elemento semântico do HTML5 usado para representar conteúdo introdutório ou links de navegação.

Aplicações comuns:

Exemplos de uso do elemento header

  • Cabeçalho do site
  • Cabeçalho de artigos
  • Introdução de seções
  • Cabeçalho de formulários
